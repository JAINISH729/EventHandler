🎓 Student Activity Manager

A simple React.js application that allows users to manage student activities such as adding students, marking attendance, toggling details, and deleting students.

This project is designed for beginners learning React concepts like events, conditional rendering, refs, and state management.

🚀 Features

➕ Add new students

📋 Display student list

🔍 Toggle student details

✅ Mark student as Present

❌ Mark student as Absent

🗑 Delete student by ID

⚡ Instant UI updates using React state

🧩 React Concepts Used

This project demonstrates important React concepts:

useState – Manage student data

useRef – Access input field

Props – Pass data between components

Event Handling – Handle button clicks and form submission

Conditional Rendering – Show/hide student details

List Rendering – Display students using map()

Fragments – Avoid unnecessary DOM elements

Passing Arguments in Event Handlers

📁 Project Structure src │ ├── components │ ├── StudentForm.jsx │ ├── StudentList.jsx │ └── StudentCard.jsx │ ├── App.jsx └── main.jsx



📊 How the Application Works 1️⃣ Add Student

Users enter a student name and click Add. The student is added to the list using React state.

2️⃣ Toggle Details

Click Details to show or hide student attendance options.

3️⃣ Mark Attendance

Users can mark students as:

Present

Absent

4️⃣ Delete Student

Click Delete to remove the student using their unique ID.

🎯 Learning Purpose

This project helps beginners understand how React updates the UI dynamically using:

State updates

Event-driven rendering

Component-based architecture
